Command Line Upload

You can also upload images using the Artisan command:

Demo Mode

This is a demo version. The upload functionality will show simulated responses.

Note: This demo shows the UI without actual Cloudinary integration. To use real upload functionality, you need to:

  • Configure Cloudinary credentials in .env file
  • Run Laravel server with: php artisan serve
  • Access via: http://localhost:8000
